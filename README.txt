Como usar:
1) Faça o download deste .zip e descompacte em sua máquina.
2) Substitua 'images/logo.png' pelo logo oficial (mesmo nome de arquivo ou atualize o src em index.html).
3) Substitua as imagens da galeria por fotos reais mantendo os nomes gallery1.jpg ... gallery6.jpg, ou atualize os caminhos no index.html.
4) Para hospedar rapidamente (sem usar linha de comando):
   - Netlify: crie conta, clique em "Add new site" -> "Deploy manually", arraste a pasta descompactada.
   - Vercel: crie conta, clique em "New Project" -> "Import" ou arraste os arquivos.
   - GitHub Pages: envie o conteúdo para um repositório e ative GitHub Pages nas configurações (branch main/gh-pages).
5) Para adicionar o mapa real, substitua a div .map-placeholder por um embed do Google Maps (iframe) no bloco de contato.
6) Personalize textos diretamente no index.html e estilos no style.css.
